# Forgetting

**Belief**: Forgetting is a valuable signal that reveals where foundations are weak. The system should respond to it structurally, not just by rescheduling.

## Components

1. **[Prerequisite blocking](prerequisite-blocking/)** — When you forget something, the system surfaces its foundations for review.
2. **[Skill decay](skill-decay/)** — Automation without practice causes capability loss — in both humans and agents.
3. **Signal, not failure** — A rating of 1 ("Again") is informative. It tells the system where to focus, not that the learner has failed.
4. **Structural response** — The system responds architecturally (blocking, prerequisite surfacing) rather than merely shortening the review interval.
