# Learning in context

**Belief**: Learning is most effective when embedded in real, meaningful work. Isolated flashcards are a fallback, not the primary mode.

## Components

1. **[Session model](session-model/)** — Work episodes are the natural unit of learning, not flashcard decks.
2. **[Interleaving](interleaving/)** — Mixing domains during review strengthens discrimination between concepts.
3. **Task-driven encounter** — Concepts are encountered because they're needed for the task at hand, not because a scheduler says it's time.
4. **Context strengthens memory** — "I learned this while deploying production" creates stronger traces than "flashcard #47". This is supported by [encoding specificity](https://en.wikipedia.org/wiki/Encoding_specificity_principle).

## Related beliefs
